ANGIE

Компания «Веб-сервер» развивает ANGIE высокопроизводительный и масштабируемый веб-сервер, который создан как fork Nginx и призван превзойти функциональность оригинала и стать снова востребованным веб-сервером на рынке решений кластерных программных комплексов. Среди наших продуктов: Angie PRO единственный коммерческий веб-сервер, разработка которого локализована в России. Angie Ingress Controller (ANIC) это решение для управления трафиком контейнеризированных приложений в Kubernetes. Также на нашем сайте вы сможете ознакомиться с подробной документацией.

ПОЧЕМУ ANGIE?

  • Совместимость с Nginx

    Angie PRO полностью совместим с Nginx, таким образом любой существующий пользователь Nginx может без серьезных затрат и простоя

  • Динамическая конфигурация Upstream

    Возможность управлять настройками upstream групп с помощью удобного Rest интерфейса без остановки

  • Расширенная статистика и real-time мониторинг

    Angie PRO имеет возможность полного мониторинга нагрузки сервера в режиме реального времени что

  • Активная проверка состояния проксируемых серверов
